01 — Flight Controller
STM32F405 — 168 MHz Cortex-M4

The LithCore flight controller is built around the STM32F405, a proven Cortex-M4 processor with hardware floating-point — essential for fast, stable PID loops. Every peripheral driver is written from scratch, with full register-level control.

168 MHz Cortex-M4F Hardware FPU DMA SPI / I2C / UART PWM / Timer
02 — ESC Architecture
6-Step Trapezoidal BLDC Control

LithDrive uses an STM32F051 to implement 6-step trapezoidal commutation — the gold standard for efficiency and simplicity in drone ESCs. The power stage is built from N-channel MOSFETs in a 3-phase half-bridge configuration, with back-EMF zero-crossing detection for sensorless operation.

6-step Trapezoidal Back-EMF ZCD N-channel MOSFET STM32F051 DSHOT
03 — Firmware Philosophy
Zero Black Boxes. Full Ownership.

We do not integrate third-party flight stacks. All firmware is developed in-house, giving us complete visibility into timing, determinism, and failure modes. This is the only way to build a truly trustworthy aerial system.

Bare-metal C FreeRTOS HAL abstraction Deterministic loops
04 — Integrated Design
System Co-Design from Day One

Hardware and firmware are co-designed. Interface protocols, timing constraints, and power budgets are resolved at the board level before a single line of firmware is written. This eliminates integration debt and produces cleaner, more reliable systems.

HW/FW co-design Power budgeting Signal integrity Thermal analysis

Vision

"Most drone systems are assembled from vendor components. Aerolith is different — every layer is engineered, owned, and understood. This is how you build systems that perform at the edge."

Our mission is to create a vertically integrated aerial technology stack — where hardware decisions inform firmware design, firmware informs control algorithms, and control algorithms inform the mechanical platform.

01
Engineering Integrity

We design systems we fully understand. No hidden layers, no vendor lock-in, no unexplained behavior. If we can't explain it, we redesign it.

02
Vertical Integration

Silicon to airframe. Schematic to flight. We own the complete stack, which means we control quality, performance, and the roadmap.
